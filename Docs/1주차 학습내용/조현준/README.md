### 3-2. WebRTC 기본 코드 구현

전체적인 코드 구현은 Cursor AI IDE를 활용하여 설계하였다.

WebRTC와 관련된 기초 이론에 대한 내용은 Youtube 영상 및 블로그 등을 참고하여 학습하였으나, 코드에 대한 깊은 이해가 없어 디버깅을 하지 못하는 문제가 있다.

향후 코드 구현 및 디버깅을 위해 추가적인 학습이 필요하다. (네트워크 통신 관련 CS 지식, 소켓 통신, React useEffect 등)
